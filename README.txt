Обозначения QtWidgets в форму mainwindow.ui:
- le_get_photo - LineEdit, в которой отображается путь до папки, откуда будут браться фотографии для сортировки
- le_save_photo - LineEdit, в которой отображается путь до папки, в которую будут сохраняться отсортированные фото
- pbtn_get_photo - PushButton, при нажатии на которую открывается файловое диалоговое окно для выбора каталога с фотографиями, которые будут сортироваться
- pbtn_save_photo - PushButton, при нажатии на которую открывается файловое диалоговое окно для выбора каталога, в который будут сохраняться фотографии
- rbtn_delete_dublicates, rbtn_keep_dublicates - RadioButton используеются для выбора оставлять дубликаты или сразу удалять их
- label_is_delete_duplicates, label_path_nosort_catalog, label_path_sorted_catalog - Label вспомогательные элементы с текстом
- pbtn_start, pbtn_stop, pbtn_finish - PushButton кнопки управления процессом сортировки: начать сортировку, остановить, прервать полностью. Кнопка остановить после нажатия меняет свое название на "Продолжить"
- progress_bar - ProgressBar, который отображает прогресс сортировки фотографий
- te_logger - TextEdit используется для отображения логов программы
